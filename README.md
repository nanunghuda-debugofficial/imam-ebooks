## 📄 Cara Convert ke PDF

### Opsi 1: Markdown → PDF (Recommended)
```bash
# Install pandoc
sudo apt install pandoc

# Convert single ebook
pandoc EBOOK-MINI-MVP-TEST.md -o EBOOK-MINI-MVP-TEST.pdf --pdf-engine=xelatex

# Convert all
for f in ebook-*/*.md; do
  pandoc "$f" -o "${f%.md}.pdf" --pdf-engine=xelatex
done
```

### Opsi 2: Google Docs
1. Copy-paste konten ke Google Docs
2. File → Download → PDF Document

### Opsi 3: Online Tool
- https://www.markdowntopdf.com/
- https://md2pdf.netlify.app/
